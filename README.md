## Setup

Install `yarn` for macOS
```
brew install yarn
```
https://yarnpkg.com/lang/en/docs/install/

Run yarn
```
yarn
```

## Running

Run locally:
```
gulp serve
```

Build production version:
```
gulp
```

Run production version:
```
gulp serve:prod
```

Run unit tests:
```
gulp test
```
